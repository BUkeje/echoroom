# 🎧 EchoRoom

EchoRoom is a real-time album listening party web app where multiple users can join the same room and listen to synced music together from anywhere.

---

# Features

- Real-time room system
- Live chat with WebSockets
- Synced music playback
- Automatic track progression
- Dark modern UI
- Multi-user listening experience
- Full-stack JavaScript + Python project

---

# Tech Stack

## Frontend

- React
- JavaScript
- HTML
- CSS
- Vite

## Backend

- Python
- FastAPI
- WebSockets
- Uvicorn
